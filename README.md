#Personal Portfolio | R Dhashwin Kennedy
This repository contains the source code and assets for my personal portfolio website, which showcases my projects, skills, and professional journey as a developer.

**Live Demo: [dhashwinkennedy.me](https://dhashwinkennedy.me)**

## Overview

This website is designed to be a central hub for my professional identity. It features a responsive design, interactive UI components, and a detailed look at my work in full-stack development, specifically focusing on the MERN stack and automation.

##Key Features

- **Project Showcase**: A curated list of web applications and software tools.

- **Technical Skills:** Deep dive into my expertise in React, Node.js, Python, and more.

- **Responsive Design:** Optimized for seamless viewing across desktops, tablets, and mobile devices.

- **Contact Integration:** Direct links to my professional social profiles.

## 🛠️ Tech Stack

The project is built using modern web technologies:

- Frontend: React.js / Vite

- Styling: Tailwind CSS / CSS3
